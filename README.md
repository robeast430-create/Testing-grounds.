# Neural Agent

An advanced AI agent with persistent memory, autonomous thinking, task execution, and a kill switch.

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
python -m neural_agent.main
```

## Usage

- Chat with the agent
- Feed it information with `learn <info>`
- Give it tasks with `do <task>`
- Kill it with `KILL` (activates kill switch)
- Exit with `quit`
